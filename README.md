## How to Add an Article

1. Open `client/src/data/articles.ts`
2. Add a new object to the `articles` array
3. Required fields: `id`, `slug`, `title`, `category`, `categorySlug`, `metaDescription`, `excerpt`, `thumbnail`, `altText`, `datePublished`, `dateModified`, `content`
4. The homepage and category pages update automatically

## How to Update an Article

1. Find the article by its slug in `client/src/data/articles.ts`
2. Edit the `content` field (supports Markdown)
3. Update `dateModified` to today's date
